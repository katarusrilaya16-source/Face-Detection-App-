A real-time Face Detection Application developed using Python and OpenCV that detects human faces through a webcam or image input. This project uses computer vision techniques to identify and highlight faces accurately with bounding boxes.

🚀 Features
Real-time face detection using webcam
Detects multiple faces simultaneously
Fast and efficient performance
Simple and user-friendly interface
Built using OpenCV and Python
🛠️ Technologies Used
Python
OpenCV
NumPy
📸 Output
Detects and highlights faces through webcam feed
Works on images and live video streams
📈 Future Enhancements
Face recognition feature
Mask detection
Attendance system integration
Emotion detection
